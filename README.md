# HR-Employee-Attrition-Analysis
Analyse des données RH sur l’attrition des employés à partir du dataset IBM HR Analytics. Ce projet inclut :  Exploration et nettoyage des données  Visualisation des tendances (âge, rôle, satisfaction, salaire, etc.)  Modélisation prédictive de l’attrition (classification)  Tableaux de bord interactifs pour la prise de décision RH.

Structure du projet
Table principale : HR_PROJET 

Scripts SQL :

create_tables.sql → création de la table et des colonnes

import_data.sql → import du CSV et vérifications

queries_analysis.sql → requêtes analytiques (KPI RH)

views_dashboards.sql → vues prêtes pour Power BI/Tableau

Résultats
Taux d’attrition global et par département

Attrition par rôle et par satisfaction

Revenu moyen par rôle

Ancienneté moyenne par département
